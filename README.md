#Nodejs workshop

## How to install nodejs?
```bash
sudo apt-get install nodejs #For linux users

brew install nodejs #For os x users
```


## Installing test framework

```bash
$ npm install -g mocha
```

```bash
$ npm install chai
```

## Running test suite

```
$ mocha
```
